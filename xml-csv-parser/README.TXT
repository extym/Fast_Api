ТРЕБОВАНИЯ:
Python 3.9+

УСТАНОВКА:
1. Закинуть скрипт на севрер
2. Создать venv и установить необходимые библиотеки
2.1. Открыть консоль сервера
2.2. Ввести 'cd /путь/до/папки/со/скриптом' (пример: Вы закинули папку скрипта в директорию /var/, значит, Вы прописываете: 'cd /var/xml-csv-parser')
2.3. Вводите 'python3 -m venv .venv' и ждёте выполнения.
2.4. Вводите 'pip3 install -r req.txt'
3. Готова

УСТАНОВКА ЗАПЛАНИРОВАННОГО ЗАПУСКА: (пример: crontab)
Скрипт поддерживает выбор параметров из консоли:
Пример: path/to/folder> python3 main.py yandex csv

-Первый параметр - название маркетплейса (Выбрать одно из двух: avito, yandex)
-Второй параметр - формат исходных и выходных файлов (Выбрать одно из двух: xml, csv)
    Примечание: У яндекса выходной файл всегда будет в формате yml.

Пример команды для crontab: '0 23 * * * cd /path/to/folder/ && .venv/bin/python main.py avito xml

РЕЗУЛЬТАТЫ:
Таблицы сохраняются в папке /var/www/html

ЛОГИРОВАНИЕ:
Данные об ошибках и непройденных товарах хранятся в папке logs.
